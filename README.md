### 介绍

此模板是一款基于前端UI框架 [Layui](https://layui.dev) 生成的后台管理模板，此模板根据当前最新版本(2025-11-25)的Layui框架实现。主要针对喜欢Layui的后端开发者和前端开发者以及针对只掌握少量前端知识(HTML/CSS/JS)但是想快速构建后台管理系统的后端开发人员。
